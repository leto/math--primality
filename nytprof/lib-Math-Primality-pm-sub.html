99=head1 DESCRIPTION
100
101Math::Primality implements is_prime() and next_prime() as a replacement for Math::PARI::is_prime(). It uses the GMP library through Math::GMPz. The is_prime() method is actually a Baillie-PSW primality test which consists of three steps:
102
103=over 4
104
105=item * Check N for small prime divisors p < 1000
106
107=item * Perform a strong Miller-Rabin probable prime test (base 2) on N
108
109=item * Perform a strong Lucas-Selfridge test on N (using the parameters suggested by Selfridge)
110
111=back
112
113At any point the function may return as definitely composite. If not, N has passed the strong Baillie-PSW test and is either prime or a strong Baillie-PSW pseudoprime. To date no counterexample (Baillie-PSW strong pseudoprime) is known to exist for N < 10^15. Baillie-PSW requires O((log n)^3) bit operations. See L<http://www.trnicely.net/misc/bpsw.html> for a more thorough introduction to the Baillie-PSW test. Also see L<http://mpqs.free.fr/LucasPseudoprimes.pdf> for a more theoretical introduction to the Baillie-PSW test.

271=head2 is_strong_lucas_pseudoprime($n)
272
273Returns true if $n is a strong Lucas-Selfridge pseudoprime, false otherwise. The variable $n should be a Perl
274integer or a Math::GMPz object.
275
276 if ( is_strong_lucas_pseudoprime($n) ) {
277 # it's a strong Lucas-Selfridge pseudoprime
278 } else {
279 # not a strong Lucas-Selfridge psuedoprime

281 }

283=head3 Details
284
285If we let
286
287=over 4
288
289=item * $D be the first element of the sequence 5, -7, 9, -11, 13, ... for which ($D/$n) = -1. Let $P = 1 and $Q = (1 - $D) /4

291=item * U($P, $Q) and V($P, $Q) be Lucas sequences

293=item * $n + 1 = $d * 2^$s + 1

295=back

297Then a strong Lucas-Selfridge pseudoprime is an odd, non-perfect square number $n with that satisfies either
298
299=over 4

301=item * U_$d = 0 mod $n
302
303=item * V_($d * 2^$r) = 0 mod $n, for $r = 0, 1, ..., $s-1
304
305=back
306
307=head3 Notes
308
309($d/$n) refers to the Legendre symbol.
310The tuple ($D, $P, $Q) is determined by the helper function _find_dpq_selfridge().
311$d and $s are determined by the helper function _find_s_d().

313=cut
